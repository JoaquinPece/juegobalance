# 📊 Clasificador de Balance - Juego Educativo de Contabilidad

Un juego interactivo para aprender a clasificar elementos del balance contable, diseñado especialmente para estudiantes de 2º de Bachillerato.

## 🎮 Características

- **Modo Examen**: 10 preguntas con calificación final
- **Modo Práctica**: Preguntas ilimitadas para mejorar
- **30 elementos del balance** de nivel bachillerato
- **Sistema de rachas** para motivar el aprendizaje
- **Tips educativos** automáticos en cada error
- **Temporizador opcional** para mayor desafío
- **Música de fondo** (Himno de la Alegría de Beethoven)
- **Responsive**: Compatible con móviles y PC
